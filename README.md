# dotfiles

## Install
1. move to home directory
```sh
cd ~
```

2. clone this repository at home directory
```sh
git clone https://github.com/macchanism/dotfiles.git
```

3. permition setting
```sh
cd dotfiles
chmod +x install.sh
```

4. install dotfiles
```sh
./install.sh
```
